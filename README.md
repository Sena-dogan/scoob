## Getting Started
Get packages
```bash
flutter pub get
```

Run the app
```bash
flutter run
```

## Posible Errors
If you dont have generated files, you can generate them with this command
```bash
flutter pub run build_runner build
```

and of cousres if you have any error, you can clean the project with this command
```bash
flutter clean 
```
best problem solver :)
